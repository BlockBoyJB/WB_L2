Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Программа выведет:
2
1

Вывод обусловлен различием в функциях.
В случае с test x объявлен как именнованный результат (x в сигнатуре функции). 
В этом случае defer функции выполняются до возврата, поэтому defer функция "успевает" увеличить значение

В случае с anotherTest x объявлен и явно возвращается из функции (а не как простой `return` в test)
defer функция вызывается уже после возврата, поэтому не увеличивает итоговое значение

defer используется для отложенного вызова функций. Работают по принципу стека, т.е LIFO
```
